## 開始使用

### 先前準備

確認 node version >= 14.0

### 安裝

1. 複製此 repository

```sh
    git clone https://github.com/Cazanna-Tolean/weather_app_TW.git
```

2. 安裝 NPM 套件

```sh
    npm install
```

### 使用

- 開啟 repository

```sh
npm start
```

### 部署

- 藉由 gh-pages

```sh
npm run depoly
```
